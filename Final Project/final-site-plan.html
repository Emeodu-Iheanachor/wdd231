Tech Skills Hub Nigeria - Project Site Plan

1. Site Purpose

The purpose of this website is to provide a structured and beginner-friendly platform where users can learn about technology skills, access learning resources, and discover opportunities in the tech industry.

2. Target Audience

  • Students (secondary and university level)
  • Beginners interested in tech careers
  • Self-learners looking for guidance

3. Site Map (Structure)

Pages:

  • Home
    • Learning Resources
    • Opportunities

Navigation Menu:

  • Home
  • Learning Resources
  • Opportunities

4. Wireframe (Layout Design)

Home Page Layout

LOGO | Navigation Menu
HERO SECTION (Heading + Intro + Button)
Tech Fields Section (3–4 cards)
Call to Action (Start Learning)
Footer (Contact, social links)

Learning Resources Page

Header + Navigation
Page Title
Resource Cards (Course Name, Platform, Link)
Footer

Opportunities Page

Header + Navigation
Page Title
Opportunity Cards (Title, Type, Deadline)
Footer

5. Color Scheme

  • Primary Color: #0A66C2 (Blue)
  • Secondary Color: #0F172A (Dark Navy)
  • Accent Color: #22C55E (Green)
  • Background: #F8FAFC (Light Gray)
  • Text Color: #111827 (Dark Gray)

6. Typography

  • Headings: Poppins (or Arial fallback)
  • Body Text: Roboto / Open Sans
  • Fallback: sans-serif

7. Features & Functionality

  • Responsive design (mobile + desktop)
  • Navigation menu
  • Dynamic content using JavaScript + JSON
  • External links to resources
  • Simple animations (hover effects, transitions)

8. Content Strategy

Home Page: Intro to tech skills, short descriptions of fields (Frontend, Backend, UI/UX)

Learning Resources: Static + dynamic list of courses, tools, tutorials

Opportunities: Data from JSON or API (Internships, Scholarships, Events)

10. Accessibility Considerations

  • Proper heading structure (H1–H3)
  • Alt text for images
  • Good color contrast
  • Keyboard navigation support

11. Performance Considerations

  • Optimized images
  • Minimized CSS/JS
  • Lazy loading (optional)
